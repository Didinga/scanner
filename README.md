# 🐍 Dark Scanner v1.0

> "Not just code. A mindset."

A minimal, cinematic port scanner built in pure Python.
Dark aesthetic. Clean output. Zero dependencies.

## Usage

```bash
python scanner.py
```

## Target

Uses `scanme.nmap.org` — a legal, public server designed for testing.
Do NOT use on targets you don't own or have permission to scan.
